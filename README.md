### 简介

快速搭建中后台(hpaPaaS平台)数据驱动、可视化编辑的常用组件库

### 快速开始

1. 安装:

```bash
npm install --save yh5
```

2. 添加至你的vue应用:

```javascript
import Vue from 'vue'
import Yh5 from 'yh5/lib'
import * as components from 'yh5/lib/components'

Vue.use(Yh5, {
  components
})
```
